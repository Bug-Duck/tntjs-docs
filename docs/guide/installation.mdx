# Installation

TNTjs does not require any installation - yes you heard that right - it's a **pure runtime** framework, just like jQuery.

To get started with TNTjs is simple: just import the TNTjs module into your JavaScript code!

```js title="App.js"
import TNTApp from "https://cdn.jsdelivr.net/npm/tntjs@latest/dist/src/index.js";

// your code here
```

### Plain text version

Or if you prefer a plain-text version of the boilerplate code:

```html title="index.html"
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>TNT App</title>
  </head>
  <body>
    <div id="root">
      <!-- debug plugin enabled -->
      <tnt-debug></tnt-debug>
      <t-for data="link in links">
        <div>
          <v data="link">Loading link...</v>
        </div>
      </t-for>
    </div>
    <script type="module" src="./App.js"></script>
  </body>
</html>
```

```js title="App.js"
import TNTApp from "https://cdn.jsdelivr.net/npm/tntjs@latest/dist/src/index.js";
import DebugPlugin from "https://cdn.jsdelivr.net/npm/tntjs@latest/dist/src/plugins/debug/index.js";

window.onload = () => {
  // create a new TNT app
  const app = new TNTApp(document.getElementById("root"), () => {
    app.addPlugins([DebugPlugin]);  // add a plugin
    setTimeout(() => {
      app.variables.links = [  // variable re-assignment
        ...app.variables.links,
        { link: "https://new.org", target: "_blank" },
      ];
    }, 5000);
  });
  // initial data
  app.data({
    links: [
      "https://google.com",
      "https://bing.com",
      "https://yahoo.com"
    ],
  });
};
```

## Installing with Nodejs

Of course, if you insist to install TNTjs for whatever reason, we do actually provide a NPM package. Simply install it using your favorite package manager:

<Tabs>
<TabItem value="npm" label="NPM" default>

```bash
$ npm i tntjs
```

</TabItem>
<TabItem value="yarn" label="Yarn">

```bash
$ yarn add tntjs
```

</TabItem>
<TabItem value="pnpm" label="PNPM">

```bash
$ pnpm i tntjs
```

</TabItem>
</Tabs>

Then you can use TNTjs with Nodejs at your own demand.
