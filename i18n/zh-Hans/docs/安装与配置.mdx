# 安装与配置

### 通过包管理器安装
虽然tntjs是一个原生的前端框架，但你仍然可以通过包管理器安装他，有三种方式可供安装
<Tabs>
<TabItem value="npm" label="NPM" default>

```bash
$ npm i tntjs
```

</TabItem>
<TabItem value="yarn" label="Yarn">

```bash
$ yarn add tntjs
```

</TabItem>
<TabItem value="pnpm" label="PNPM">

```bash
$ pnpm i tntjs
```

</TabItem>
</Tabs>
